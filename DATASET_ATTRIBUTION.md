# Dataset Attribution

- Dataset name: UAE
- Canonical URL: https://universe.roboflow.com/addinguae/uae-zcfqj
- Dataset license: CC BY 4.0

This project filters the original multiclass annotations into one detection target representing the complete visible license plate. OCR digits and letters, expiration marks, emirate identifiers, and plate-style annotations are excluded from the target labels.

The CC BY 4.0 attribution applies to the source dataset. It does not establish or imply a license for this repository's code.

The local raw export metadata refers to a different Roboflow workspace URL than the canonical attribution URL above. That discrepancy is recorded as source metadata provenance; the two URLs are not asserted to be identical.
